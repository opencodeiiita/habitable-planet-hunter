# Contributing Guidelines

Thank you for contributing to this repository!

## What You Have To Do

### Step 1
Create a `.txt` file with **your GitHub username** as the file name.

### Step 2
Inside the file, add the following details:

GitHub Username: your-github-username  
Real Name: Your Full Name  
Year of College: 1st / 2nd / 3rd / 4th  
College Name: Your College or University Name

## Example

File name:
octocat.txt

File content:
GitHub Username: octocat  
Real Name: Mona Lisa  
Year of College: 3rd  
College Name: GitHub University

## Final Step
Commit your changes and push or create a pull request.

Thank you for contributing!
